# Descripcion 

Este preoyecto consiste en el desarrollo de un sistema de gestion para un servicio de delivery implementado en Python como Trabajo Final Integrador de la Asignatura Algoritmos y Estructura de Datos.

El sistema funciona mediante una interfaz por consola y permite administrar pedidos de clientes, calcular automaticamente los importes, gestionar diferentes formas de pago, aplicar promociones y registrar informacion relevante para el funcionamiento del servicio.

Durante su desarrollo se aplicaron los contenidos aprendidos en la materia,haciendo uso de estructuras de control, finciones, validaciones, estructuras de datos y modularizacion para contruir una solucion optimizada y funcional.

---

# Funcionalidades

El sistema permite:

- Registrar Nuevos pedidos.
- Registrar multiples productos por pedido.
- Calccular automaticamente el subtotal y el total de la compra.
- Calcular el costo de envio segun la zona seleccionada.
- Aplicar promociones ssegun el dia de la semana.
- Gestionar clientes frecuentes con beneficios de envio.
- Registrar distintos medios de pago.
- Calcular pagos en cuotas.
- Otorgar regalos promocionales segun el importe del pedido.
- Vizualizar los pedidos registrados.
- Acceder a un menu destinado a repartidores.
- Registrar informacion basica de repartidores.
